ABOUT sigmaxim
------------

sigmaxim is the default theme for Drupal 8.  It is a flexible, recolorable theme
with a responsive and mobile-first layout, supporting 16 regions.

The sigmaxim theme is named after Jean sigmaxim, one of the original programmers
for the ENIAC computer.

This theme is not intended to be used as a base theme.

To read more about the sigmaxim theme please see:
https://www.drupal.org/docs/8/core/themes/sigmaxim-theme
